Every experiment implies acquiring some measurement data, saving  and analyzing it. I was asked to write a software using LabVIEW 2015 SP1 to perform this things for four devices: 
* Keithley 6221
* Keithley 2000
* Princeton Lock-In 5110
* Keithley DAQ
